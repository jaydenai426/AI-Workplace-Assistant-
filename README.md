## 📌 Project Overview
Aidesk is designed to streamline productivity with a unified workspace for tasks, meetings, email, and research.  
This first slice delivers a polished foundation with essential UX, onboarding, and SEO scaffolding.

## ✨ Features (First Slice)
- **[Dark Mode](ca://s?q=Explain_dark_mode_in_web_apps)** – End-to-end implementation with pre-hydration boot script to prevent flash.  
- **[Global Command Palette](ca://s?q=What_is_a_command_palette)** (⌘K) – Quickly jump to tools, start chat/research, or sign out.  
- **[Onboarding Checklist](ca://s?q=Onboarding_checklist_in_apps)** – Tracks first use of each tool via activity events and hides once complete.  
- **[Starter Prompts](ca://s?q=Starter_prompts_in_apps)** – Empty states for Email and Task Planner, plus one-click “sample notes” in Meetings.  
- **[Responsive Header](ca://s?q=Responsive_header_design)** – Mobile view shortens to “Aidesk”; desktop shows ⌘K hint.  
- **[SEO Basics](ca://s?q=SEO_basics_for_websites)** – `robots.txt` and `/sitemap.xml` server route pointing at the production domain.
